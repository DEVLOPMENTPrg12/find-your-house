# FindYourHouse

FindYourHouse is a modern web application built with **React.js** and **Firebase**, designed to help users search, filter, and book rental houses across different cities in Morocco.  

## Features
- **Search by city** – Quickly find houses in your desired location.
- **Filter by price and features** – Narrow down options based on your budget and preferences.
- **House details** – View images, room count, amenities, and interior photos.
- **Add to Cart / Book** – Add houses to cart for later or directly book via a reservation form.
- **User Authentication** – Register and login to manage bookings securely.
- **Contact Page** – Easily reach out to the owner or support.
- **Responsive Design** – Works smoothly on mobile, tablet, and desktop.

## Tech Stack
- **Frontend:** React.js, Bootstrap
- **Backend / Database:** Firebase (Authentication, Firestore)
- **Hosting:** Firebase Hosting (optional: GitHub Pages, Vercel, Netlify)
